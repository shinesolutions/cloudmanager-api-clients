This is the API

deleteEnvironment

Delete environment

URL

https://cloudmanager.adobe.io/api/program/{programId}/environment/{environmentId}

HTTP Method

DELETE

Response Type

Environment

Parameters

  • Path: programId String

    Identifier of the application

  • Path: environmentId String

    Identifier of the environment

  • Header: x-gw-ims-org-id String

    IMS organization ID that the request is being made under.

  • Header: Authorization String

    Bearer [token] - An access token for the technical account created through integration with Adobe IO

  • Header: x-api-key String

    IMS Client ID (API Key) which is subscribed to consume services on console.adobe.io

downloadLogs

Download environment logs

URL

https://cloudmanager.adobe.io/api/program/{programId}/environment/{environmentId}/logs/download

HTTP Method

GET

Response Type

Parameters

  • Path: programId String

    Identifier of the program

  • Path: environmentId String

    Identifier of the environment

  • Query: service String

    Name of service

  • Query: name String

    Name of log

  • Query: date String

    date for which log is required

  • Header: x-gw-ims-org-id String

    IMS organization ID that the request is being made under.

  • Header: Authorization String

    Bearer [token] - An access token for the technical account created through integration with Adobe IO

  • Header: x-api-key String

    IMS Client ID (API Key) which is subscribed to consume services on console.adobe.io

  • Header: Accept String

    Specify application/json in this header to receive a JSON response. Otherwise, a 307 response code will be returned with a Location header.
