This is the API

getEnvironmentVariables

List the user defined variables for an environment (Cloud Service only).

URL

https://cloudmanager.adobe.io/api/program/{programId}/environment/{environmentId}/variables

HTTP Method

GET

Response Type

variableList

Parameters

  • Path: programId String

    Identifier of the program

  • Path: environmentId String

    Identifier of the environment

  • Header: x-gw-ims-org-id String

    IMS organization ID that the request is being made under.

  • Header: Authorization String

    Bearer [token] - An access token for the technical account created through integration with Adobe IO

  • Header: x-api-key String

    IMS Client ID (API Key) which is subscribed to consume services on console.adobe.io

patchEnvironmentVariables

Modify multiple environment variables (Cloud Service only). To delete a variable, include it with an empty value.

URL

https://cloudmanager.adobe.io/api/program/{programId}/environment/{environmentId}/variables

HTTP Method

PATCH

Response Type

variableList

Parameters

  • Path: programId String

    Identifier of the program

  • Path: environmentId String

    Identifier of the environment

  • Header: x-gw-ims-org-id String

    IMS organization ID that the request is being made under.

  • Header: Authorization String

    Bearer [token] - An access token for the technical account created through integration with Adobe IO

  • Header: x-api-key String

    IMS Client ID (API Key) which is subscribed to consume services on console.adobe.io

  • Header: Content-Type String

    Must always be application/json

  • Body: body List(Variable)

    The list of variables to add, modify, or remove. It is not necessary to send variables here which are not changing.
