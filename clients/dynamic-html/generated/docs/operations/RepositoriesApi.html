This is the API

getRepositories

Lists all Repositories in an program

URL

https://cloudmanager.adobe.io/api/program/{programId}/repositories

HTTP Method

GET

Response Type

repositoryList

Parameters

  • Path: programId String

    Identifier of the program

  • Header: x-gw-ims-org-id String

    IMS organization ID that the request is being made under.

  • Header: Authorization String

    Bearer [token] - An access token for the technical account created through integration with Adobe IO

  • Header: x-api-key String

    IMS Client ID (API Key) which is subscribed to consume services on console.adobe.io

getRepository

Returns an repository by its id

URL

https://cloudmanager.adobe.io/api/program/{programId}/repository/{repositoryId}

HTTP Method

Response Type

Parameters

    Identifier of the program

  • Path: repositoryId String

    Identifier of the repository

  • Header: x-gw-ims-org-id String

    IMS organization ID that the request is being made under.

  • Header: Authorization String

    Bearer [token] - An access token for the technical account created through integration with Adobe IO

    IMS Client ID (API Key) which is subscribed to consume services on console.adobe.io
