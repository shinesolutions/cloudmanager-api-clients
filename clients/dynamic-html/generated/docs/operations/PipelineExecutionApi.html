This is the API

advancePipelineExecution

Post to this url in order to advance the current pipeline execution, if paused and waiting for user interaction. Link is present in output only in that case. The input depends on the actual reason for which the pipeline execution stopped.

URL

https://cloudmanager.adobe.io/api/program/{programId}/pipeline/{pipelineId}/execution/{executionId}/phase/{phaseId}/step/{stepId}/advance

HTTP Method

PUT

Response Type

Parameters

  • Path: programId String

    Identifier of the program.

  • Path: pipelineId String

    Identifier of the pipeline

  • Path: executionId String

    Identifier of the execution

  • Path: phaseId String

    Identifier of the phase

  • Path: stepId String

    Identifier of the step

  • Header: x-gw-ims-org-id String

    IMS organization ID that the request is being made under.

  • Header: Authorization String

    Bearer [token] - An access token for the technical account created through integration with Adobe IO

  • Header: x-api-key String

    IMS Client ID (API Key) which is subscribed to consume services on console.adobe.io

  • Header: Content-Type String

    Must always be application/json

  • Body: body Object(object)

    Input for advance. See documentation for details.
