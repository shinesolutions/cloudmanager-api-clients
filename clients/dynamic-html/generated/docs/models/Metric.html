Metric

  • id : String
    KPI result identifier
  • severity : String
    Severity of the metric
    Enum:
    critical
    important
    informational
  • passed : Boolean
    Whether metric is considered passed
  • override : Boolean
    Whether user override the failed metric
  • actualValue : String
    Expected value for the metric
  • expectedValue : String
    Expected value for the metric
  • comparator : String
    Comparator used for the metric
    Enum:
    GT
    GTE
    LT
    LTE
    EQ
    NEQ
  • kpi : String
    KPI identifier
